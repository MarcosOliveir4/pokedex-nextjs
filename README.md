# 📍 Pokédex Next.js

Este é um projeto **Next.js** de uma **Pokédex**, criado com [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app), com foco em qualidade, testes e estilo moderno.

## 🚀 Visão Geral

Pokédex interativa com:

- 📋 Listagem de Pokémons
- 🔎 Filtro por tipo
- 🧠 Detalhes de cada Pokémon
- 🧪 Testes com **Vitest**
- 🎨 Estilização com **TailwindCSS**

## 🛠️ Pré-requisitos

Antes de iniciar, tenha instalado:

- Node.js **v24.13.0**
- npm ou outro gerenciador (yarn / pnpm / bun)

---

## 🚀 Como Rodar o Projeto

### 1. Clone o repositório

```bash
git clone https://github.com/MarcosOliveir4/pokedex-nextjs.git
cd pokedex-nextjs
```

### 2. Instale dependências

```bash
npm install
```

### 3. Inicie o servidor de desenvolvimento

```bash
npm run dev
```

## 🧪 Testes e Coverage

Este projeto usa Vitest para testes.

A cobertura de testes é coletada e enviada automaticamente via GitHub Actions para o Qlty (visualizada nos badges no topo).
